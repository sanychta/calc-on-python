Простой консольный калькулятор с поддержкой базовых арифметических операций.

- Сложение (`+`)
- Вычитание (`-`)
- Умножение (`*`)
- Деление (`/`)
- Остаток от деления (`%`)
- Возведение в степень (`^`)
- Обработка ошибок (деление на ноль, неверный ввод)
- Возможность выхода в любой момент

Для запуска требуется наличие установленного Python 3.x

Запуск: python calc.py
